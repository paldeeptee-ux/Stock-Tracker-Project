# Stock Tracker Project

A simple Python-based command-line tool to track stock investments. It calculates total investments based on hardcoded stock prices and saves the final output to a text file.

## Features
- Interactive Command-Line Interface (CLI).
- Automatically converts stock names to uppercase (handles case sensitivity).
- Calculates individual investment per stock and the overall portfolio total.
- Exports the final tracking result automatically to a `result.txt` file.

## How to Run
1. Make sure you have Python installed.
2. Clone or download this repository.
3. Run the script using your terminal:
```bash
   python stock_tracker.py
